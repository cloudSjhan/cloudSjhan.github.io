Support — AnySearch

Getting Help

If you encounter any issues or have questions about AnySearch, we’re here to help.

Common Questions

How do I grant file access?

When you first launch AnySearch, it will ask for permission to access your home directory. Click “Allow” in the system dialog. If you accidentally denied access, go to System Settings → Privacy & Security → Files and Folders, and enable access for AnySearch.

My files aren’t showing up in search results

  • Make sure the directory containing your files has been granted access.
  • Go to Settings and click “Rebuild Index” to re-scan your files.
  • Hidden files are excluded by default. Enable “Scan Hidden Files” in Settings if needed.

How do I restore my purchase?

Open the app, trigger the paywall (or go to Settings), and click “Restore Purchases”. Your purchase is linked to your Apple ID and can be restored on any Mac.

How do I re-open the main window?

If you closed the main window, use the menu bar: Window → Show Main Window (⌘0), or use your configured hotkey (default: ⌥Space).

How do I change the global hotkey?

Go to Settings → Hotkey and click the recorder to set a new keyboard shortcut.

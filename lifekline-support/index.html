Common Questions

What is 知运 LifeKline?

知运 LifeKline is an iOS app that helps users enter Bazi information, generate life fortune trend analysis, read daily fortune, and receive decision divination suggestions.

All generated content is for lifestyle and entertainment reference only. It should not be treated as medical, legal, financial, psychological, or other professional advice.

How do I generate an analysis?

Open the app, go to the analysis page, fill in or select your Bazi / birth information, then tap the generate button. The app will create a fortune trend and AI reading based on the selected analysis mode.

What is the difference between built-in AI and custom API?

Built-in AI uses the LifeKline analysis service and does not require you to provide your own API Key.

Custom API lets you use your own OpenAI-compatible model provider. You can configure:

  • Provider
  • Base URL
  • Model
  • API Key

Your custom API Key is saved locally in the iOS Keychain.

Why does custom API fail?

Please check:

  • Base URL is correct and includes the provider’s API version path when required
  • Model name is supported by your provider
  • API Key is valid and has available quota
  • The provider supports an OpenAI-compatible /chat/completions API
  • Your network connection is available

How do I restore my purchase?

Open Settings in the app and tap “Restore Purchases”. Your purchase is linked to your Apple ID and can be restored through Apple’s StoreKit system.

What is included in Pro?

Pro unlocks built-in AI usage beyond the free trial limit and enables result image export / sharing for supported features such as daily fortune and decision divination.

Why does the app show a 404 or API error?

This usually means the selected analysis endpoint is unavailable, the backend service is not updated, or the custom API Base URL is incorrect. If you are using built-in AI mode and still see this issue, please contact support with the error message and app version.

Can I delete my history?

The app stores recent analysis history locally on your device. You can delete history entries from the app where supported, or remove all local app data by deleting the app from your device.

Does the app provide professional advice?

No. The app provides lifestyle, entertainment, and cultural reference content based on Bazi and related traditional methods. It is not a substitute for professional advice.
